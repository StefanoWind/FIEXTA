# LiSBOA (LiDAR Statistical Barnes Objective Analysis)
*This package is registered as an NREL Software Record with identification number SWR-24-129.*

<p align="center">
  <img src="lisboa/lisboa_logo.jpg" alt="Project Logo" width="200">
</p>

Suite of codes to perform objective statistical analysis of scattered observations. 
Clone to your local machine, navigate to the location where the setup.py lives, and then:
`pip install -e .`

This will install LiSBOA as an editable Python package in the current environment.
